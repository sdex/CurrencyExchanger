# CurrencyExchanger

The app uses public APIs to load exchange rates. \
The initial balance is 1000 EUR. The balance is stored in memory and does not persist across application runs. 

This demo app is written using the following: 

- Kotlin
- JetPack Compose
- Retrofit
- OkHttp
- Koin
